# CityMetric Assets — Image Placeholder Instructions

Letakkan file gambar di folder ini (`assets/`) dengan nama dan dimensi yang tepat.

## 📋 Daftar Gambar yang Dibutuhkan

### 1. **hero_city.jpg** (1400 × 280 pixels)
- **Lokasi**: Di halaman Overview (banner atas)
- **Deskripsi**: Foto udara atau malam hari dari kota metropolitan (seperti Tokyo, New York, Singapore, atau kota besar lainnya)
- **Gaya**: Kontras tinggi, cityscape dari atas, cahaya malam atau siang yang dramatis
- **Tips**: Cari di unsplash.com atau pexels.com dengan keyword: "aerial city night", "cityscape from above", "urban landscape"

### 2. **cluster_0.jpg** (400 × 260 pixels)
- **Lokasi**: Cluster "Balanced Cities" di halaman Cluster Analysis
- **Contoh kota**: Barcelona, Melbourne, Vienna
- **Deskripsi**: Foto street view atau arsitektur kota yang seimbang, elegan
- **Gaya**: Foto jalan, bangunan bersejarah, area modern yang nyaman

### 3. **cluster_1.jpg** (400 × 260 pixels)
- **Lokasi**: Cluster "Digital Leaders" di halaman Cluster Analysis
- **Contoh kota**: Tokyo, Amsterdam, San Francisco
- **Deskripsi**: Tech hub, modern buildings, tech ecosystem vibes
- **Gaya**: Arsitektur modern, startup office aesthetic, futuristik

### 4. **cluster_2.jpg** (400 × 260 pixels)
- **Lokasi**: Cluster "Established Hubs" di halaman Cluster Analysis
- **Contoh kota**: New York, London, Paris
- **Deskripsi**: Landmark ikonik, business district, kota mapan
- **Gaya**: Cityscape malam atau siang, gedung pencakar langit, landmark terkenal

### 5. **cluster_3.jpg** (400 × 260 pixels)
- **Lokasi**: Cluster "Rising Stars" di halaman Cluster Analysis
- **Contoh kota**: Seoul, Bangalore, Bangkok
- **Deskripsi**: Kota berkembang, mix tradisional-modern, growth/energy
- **Gaya**: Pertumbuhan urban, area development, campuran budaya

### 6. **cluster_4.jpg** (400 × 260 pixels)
- **Lokasi**: Cluster "Emerging Markets" di halaman Cluster Analysis
- **Contoh kota**: Jakarta, Mumbai, Manila
- **Deskripsi**: Kota berkembang yang dinamis, energi tinggi
- **Gaya**: Pasar, street life, pertumbuhan ekonomi, urban diversity

### 7. **about_banner.jpg** (1200 × 220 pixels)
- **Lokasi**: Di halaman About (banner atas)
- **Deskripsi**: Visualisasi data kota abstrak atau foto udara kota dengan filter data
- **Gaya**: Slightly desaturated, data-driven aesthetic, bisa aerial view dengan overlay data
- **Tips**: Bisa artificial/designed image atau foto dengan color grading

## 📝 Tips Mencari Gambar

- **Gratis**: unsplash.com, pexels.com, pixabay.com
- **Berbayar tapi bagus**: shutterstock.com, istock.com
- **AI Generated (jika perlu)**: perplexity.com, openai.com/dall-e
- **Pastikan dimensi tepat** atau crop/resize sesuai rasio

Setelah semua file gambar ditaruh di folder ini, dashboard akan otomatis menampilkannya tanpa perlu edit kode!
