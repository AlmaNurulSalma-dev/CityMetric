# Insights Card Background Images

Place city images here for the Insights cards luxury design.

## Image Specifications

- **Format**: JPG
- **Dimensions**: 400×300px (4:3 aspect ratio)
- **Naming**: `lowercase_city_name.jpg` (use underscores for spaces)
  - Example: `new_york.jpg`, `hong_kong.jpg`, `san_francisco.jpg`

## Required Cities

Untuk top cities sa Insights cards (Best Value, Rising Stars, Innovation Leaders):

1. **Best Value Cities** - 5 images needed:
   - zurich.jpg
   - melbourne.jpg
   - singapore.jpg
   - seoul.jpg
   - hong_kong.jpg

2. **Rising Stars** - 5 images needed:
   - shanghai.jpg
   - bangkok.jpg
   - mumbai.jpg
   - jakarta.jpg
   - bangalore.jpg

3. **Innovation Leaders** - 5 images needed:
   - new_york.jpg
   - san_francisco.jpg
   - tokyo.jpg
   - london.jpg
   - amsterdam.jpg

## Image Content

Each image should be an iconic, recognizable photo of that city:
- **Established/Mature Cities**: Landmarks, historic areas, business districts, night skylines
- **Rising/Growth Cities**: Modern development, tech districts, urban growth, energy
- **Innovation Centers**: Tech hubs, modern architecture, innovation areas

## Where to Find

- **Free stock photos**: Unsplash, Pexels, Pixabay
- **Search terms**: "{city name} skyline", "{city name} landmarks", "{city name} cityscape"
- **Optional**: Slightly desaturated or warm-toned photos work great with dark overlay

## Implementation

Images are automatically loaded when placed in this folder with correct naming. The dashboard will display them with:
- Dark gradient overlay (transparent → black)
- White text overlay with city name and metrics
- Gold rating badge
- Professional luxury card styling

If image is not found, the dashboard falls back to a solid color card design.
